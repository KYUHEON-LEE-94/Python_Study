2. 제어문
	- 어떤 조건에 의해 프로그램의 흐름을 바꾸는 기능
	조건문
		if문
	반복문
		while문
		for문
		> cf ) c언어, java에서는 do ~ while문

	1) 제어문
		프로그래밍 언어에서 실행순서를 변경하거나
		조건에 따라서 실행해야할 문장을 결정하거나
		특정한 문장을 반복해야할 때 사용함

	2) 조건문
		조건에 따라서 다른코드를 실행(분기처리) 시킬 때 사용함
	
3. if문
	if 조건식:
		if문 안의 문장(조건식이 참일때 실행)
	if문 밖의 문장(조건식이 거짓일 때 실행 / 조건식이 참일 때 실행되는 문장 다음에 실행됨)

	1) if문을 만나면 조건식을 확인한다.
	2) 만약 조건식이 True라면 if문 안으로 들어와서(들여쓰기) 문장을 실행한다
	3) 만약 조건식이 False라면 if문 안으로 들어가지 않고(들여쓰기) 다음 문장을 실행한다

4. if~else문
	if 조건식:
		if문 안의 문장(조건식이 참일 때 실행)
	else:
		else문 안의 문장(조건식이 거짓일 때 실행)

5. if ~ elif ~ else문
	if 조건식1:
		실행문1(조건식1이 참일 때 실행)
	elif  조건식2:
		실행문2(조건식1이 거짓이고  조건식2가 참일 때 실행)
	else:
		실행문3(조건식 1이 거짓이고, 조건식 2도 거짓일 때 실행)

6. if, elif, else 차이
	elif, else는 기본적으로 생략이 가능하다.
	위의 조건이 만족되지 않는 경우에 다음에 체크할 조건부분을 실행(분기)할 수 있다.

7. if ~ if, if ~ elif 차이
	if~if문의 경우 항상 두 if문의 조건을 체크함
	if~elif문의 경우 앞의 if문 조건이 만족된다면(True) 뒤에 elif문의 조건은 체크하지 않음

(ex) 
a = 4
if ~if 문의 경우
if a > 3:
	print("a가 3보다 큽니다!")
if a > 0:
	print("a가 0보다 큽니다!")

출력결과 => a가 3보다 큽니다!
	 a가 0보다 큽니다!

if ~elif 문의 경우
if a > 3:
	print("a가 3보다 큽니다!")
elif a > 0:
	print("a가 0보다 큽니다!")

출력결과 => a가 3보다 큽니다!

[예제1]
a = 6
if a > 10:
    print('a가 10보다 큽니다!')
elif a > 5:
    print('a가 10보다는 안커도 5보다는 큽니다!')
else:
    print('a가 5보다 크지 않습니다!')

# 출력 결과
a가 10보다는 안커도 5보다는 큽니다!

[예제2] 만일 a가 10보다 큰 값이 주어지면, 가장 위의 if에서 조건을 만족하므로 해당 조건문의 아래, elif와 else는 실행되지 않는다.
b = 11

if b > 10:
    print('b가 10보다 큽니다!')
elif b > 5:
    print('b가 10보다는 안커도 5보다는 큽니다!')
else:
    print('b가 5보다 크지 않습니다!')

# 출력 결과
b가 10보다 큽니다!


[예제3] 여러개의 elif를 사용하는 것도 얼마든지 가능하며 가장 위의 조건부터 순차적으로 검사함
