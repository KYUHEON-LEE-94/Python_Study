1. 연산자
	1) 산술연산자(p72)
		+
		-
		/	나눗셈, 그 결과값 자체
		//	 몫연산자
		%	나머지 연산자

	2)복합 대입연산자
			산술연산을 먼저 진행하고 그 뒤에 나온 값을 변수에 대입한다.
		+=
		-=
		*=
		/=
		%=
		**=

	3) 관계연산자(p77)
		a > b 	: a가 b보다 크니?
			맞으면 T, 틀리면 F
		a<b	: a가 b보다 작니?
			맞으면 T, 틀리면 F
		a >= b 	: a가 b보다 크거나 같니?
			맞으면 T, 틀리면 F
		a <= b	: a가 b보다 작거나 같니?
			맞으면 T, 틀리면 F
		a ==b	: a랑 b가 같니?
			맞으면 T, 틀리면 F
		a != b	: a가 b랑 같지 않니?
			맞으면 T, 틀리면 F
		
	4) 논리 연산자
		a and b 	: a도 참이고, b도 참이면
			결과는 T, 아니면 F
		a or b	:a혹은 b 둘 중 하나라도 참이면
			결과는 T, 아니면 F
		not a	: a가 참이면 F
			a가 거짓이면 T

	5) 비트연산자
		AND 연산자(&, 논리곱): 하나라도 0이면 0, 모두 1일때 1
		OR 연산자(|, 논리합): 하나라도 1이면 1, 모두 0일때 0
		XOR 연산자(^): 모두 같은 수 일때 0, 모두 다른 수 일때 1
		NOT 연산자(~): 0일때1, 1일때 0
		Shift(<<, >>): 비트를 이동시킬 횟수 지정

	6) 삼항 연산자
		참 일 때 값 if 조건식 else 거짓일때 값

2. 기본자료형
	컬렉션
		list	[]	추가, 수정, 삭제가 언제나 가능
				[]대괄호 묶어주고 ,로 구분
		tuple	()	한번 생성되고 나면 변경이 불가능
				()소괄호 묶어주고 ,(콤마)값들을 구분
		set	{}	중복된 값의 저장이 불가능		
				{]중괄호 묶어주고 ,로 구분
		dict	{키:값}	키와 값으로 한쌍이 되어서 관리
				,(콤마) 값들을 구분
